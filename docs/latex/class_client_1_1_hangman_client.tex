\doxysubsubsection{Descrizione dettagliata}
Questa classe rappresenta il client del gioco dell\textquotesingle{}impiccato.

Il client si occupa di connettersi al server e inviare le richieste di gioco rispettando il protocollo.

\begin{DoxyNote}{Nota}
Questa classe non è thread-\/safe
\end{DoxyNote}
